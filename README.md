# 🌈 Renkli Hayat — Colorful Life

> **"Renk körlüğü olan herkes, dünyayı renkleriyle görmeli."**
> *"Everyone with color blindness deserves to see the world in its full colors."*

---

## 🇹🇷 Türkçe

### Bu Proje Nedir?

**Renkli Hayat**, renk körü bireyler için geliştirilmiş, tamamen ücretsiz ve açık kaynaklı bir renk düzeltme uygulamasıdır.

Ticari amaç yoktur. Hak iddiası yoktur. Bu proje insanlığa aittir.

### Neden Yapıldı?

Dünya genelinde **300 milyonun üzerinde** insan renk körlüğü yaşamaktadır. Mevcut uygulamalar ya ücretli, ya yetersiz, ya da test amaçlıdır. Renk körü bir bireyin gerçek ihtiyacından doğan bu proje, bu boşluğu doldurmak için yapılmıştır.

### Özellikler

- 📷 **Gerçek Zamanlı Kamera Filtresi** — LMS Daltonization algoritması ile bilimsel renk düzeltme
- 🖼️ **Fotoğraf Düzeltme** — Galerindeki fotoğrafları düzeltilmiş olarak görme
- 🔍 **Renk Tanımlama** — Kamerayı tut, rengin adını öğren (Türkçe)
- ⚙️ **Kişiselleştirme** — Filtre gücü kaydırıcısı, renk türü seçimi
- 🌐 **Tarayıcıda Çalışır** — Kurulum gerektirmez

### Desteklenen Renk Körlüğü Türleri

| Tür | Açıklama |
|-----|----------|
| Deuteranopi | Kırmızı–Yeşil (en yaygın) |
| Protanopi | Kırmızı eksikliği |
| Tritanopi | Mavi–Sarı |
| Akromatopsi | Tam renk körlüğü |

## 🇬🇧 English

### What is This Project?

**Renkli Hayat (Colorful Life)** is a completely free, open-source color correction tool for people with color blindness.

No commercial intent. No ownership claims. This project belongs to humanity.

### Why Was It Built?

Over **300 million people** worldwide experience color blindness. Existing solutions are either paid, insufficient, or designed only for testing — not for real daily use. This project was born from a real need, to fill that gap.

### Features

- 📷 **Real-Time Camera Filter** — Scientific color correction using LMS Daltonization algorithm
- 🖼️ **Photo Correction** — View gallery photos with color correction applied
- 🔍 **Color Identification** — Point camera at anything, get the color name
- ⚙️ **Personalization** — Adjustable filter strength, color type selection
- 🌐 **Runs in Browser** — No installation required

### Supported Color Blindness Types

| Type | Description |
|------|-------------|
| Deuteranopia | Red–Green (most common) |
| Protanopia | Red deficiency |
| Tritanopia | Blue–Yellow |
| Achromatopsia | Complete color blindness |

## Kullanılan Algoritma / Algorithm

**LMS Color Space + Daltonization Matrix**

Machado et al. (2009) ve Fidaner et al. tarafından geliştirilen renk uzayı dönüşüm algoritması kullanılmaktadır. Bu algoritma, insan gözündeki L (uzun), M (orta) ve S (kısa) dalga boyu konilerini modelleyerek eksik olan renk bilgisini matematiksel olarak yeniden inşa eder.
